Tema 1: Etiquetas Básicas de HTML, atributos y valores

Etiquetas Básicas

HTML es un lenguaje de programación para web, sus siglas corresponden a HiperText Markup Language (en inglés) o Lenguaje de Marcas de Hipertexto (en español).
El código desarrollado en este lenguaje utiliza una serie de palabras especiales denominadas etiquetas, estas señalan el inicio y el fin de cada control específico, esta última inicia con una barra inclinada . Celaya (2014).
Estas etiquetas o marcas siguen una estructura fija establecida para el lenguaje, para indicar el inicio de una etiqueta se utiliza el siguiente formato:

< nombre_de_etiqueta >

El siguiente es un ejemplo de una etiqueta de cierre con el carácter ⁄ indicado previamente:

<⁄ nombre_de_etiqueta >
Es importante resaltar la necesidad de utilizar los caracteres “<” y “>” antes y después de cada etiqueta para que sea identificada correctamente por el navegador; por ejemplo:
La etiqueta p permite declarar un párrafo de texto, así:

<p> Texto del párrafo </p> También son conocidas como marcas, tags, directivas. Terán (2016).

Valores en HTML
