Sitio para aprendizaje de Etiquetas Básicas de HTML, atributos y valores

Este sitio web pretende mostrar material académico para la formación de estudiantes en el lenguaje HTML, aquí se mostrará información específica de las etiquetas, atributos y valores en este lenguaje.

Borrador de contenido

Etiquetas Básicas

Siendo HTML un lenguaje de programación, es necesario aclarar que es un lenguaje de marcado, o etiquetas; esto significa que cada uno de los elementos disponibles para programar se construyen por palabras especiales (etiquetas) que señalan el inicio y el fin de ese control. Celaya (2014)
Es importante resaltar la necesidad de utilizar los caracteres < y >; por ejemplo: La etiqueta p permite declarar un párrafo de texto, así:

<p> Texto del párrafo </p> También son conocidas como marcas, tags, directivas. Terán (2016).

Atributos en HTML

Son características adicionales de las etiquetas, modifica el significado de una etiqueta y puede tener múltiples valores. Terán (2016).
Ejemplo: El siguiente es una caja de texto deshabilitada a través del atributo disabled
Código: <input disabled >

Valores en HTML

Terán (2016) define a los valores como las características de los atributos, son valores que se ubican después del atributo, un signo = y están entre comillas, así:
<etiqueta atributo="valor">
