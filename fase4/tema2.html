Atributos y valores

Los atributos son elementos adicionales que están contenidos dentro de la etiqueta de apertura o inicio de un objeto HTML. Estos especifican características particulares del elemento o información adicional, siguen la siguiente estructura (ejemplo tomado de Recio(2016)):

<etiqueta atributo="valor">contenido</etiqueta>

La especificación del lenguaje incluye un conjunto global de atributos que pueden ser utilizados en todos los elementos HTML, el listado completo está disponible en: HTML Global attributes (w3schools.com). A continuación se indican algunas de las etiquetas globales más importantes:

Etiqueta Descripción
class

Especifica uno o más nombres de clases definidas en las hojas de estilo 
id

Especifica un ID (identificador) único para el elemento dentro de una página
lang Especifica el lenguaje en que se encuentra el contenido del elemento
style Especifica estilos en línea (estilos definidos directamente en el elemento HTML en lugar de una hoja de estilos)
TabIndex Especifica el orden de tabulación de un elemento

Existe un gran listado de atributos que pueden ser aplicados a ciertos elementos HTML, por ejemplo el atributo autoplay sólo puede ser utilizado con los elementos audio y video; el listado completo de atributos y los elementos en los cuales pueden ser utilizados está disponible aquí: HTML Attributes (w3schools.com)
